# BODRIN Coffee Landing

Одностраничный лендинг кофейни в стиле макета из примера. Чистый HTML и CSS (без сборки и без препроцессоров).

## Содержимое
- `index.html` — разметка страницы.
- `styles.css` — стили, палитра через CSS custom properties.
- (нет зависимостей, нет сборки)

## Как запустить локально
1. Склонируйте репозиторий: `git clone <repo-url>`
2. Откройте `index.html` прямо в браузере (двойной клик или `Open With Live Server`).

## Настройка под себя
- Замените текстовые блоки на свои тексты.
- Обновите цены/оценки/объёмы напитков в карточках.
- При необходимости добавьте реальные изображения чашек/зёрен в секции героя и меню.
- Цвета/градиенты регулируются переменными в `styles.css` (`:root`).

## Публикация на GitHub Pages
1. Закоммитьте изменения и запушьте в GitHub.
2. В настройках репозитория откройте `Pages` → Source: `Deploy from a branch`, выберите `main` и `/ (root)`.
3. Сохраните — страница будет доступна по адресу вида `https://<user>.github.io/<repo>/`.
